#Paradigm-Crawler

Paradigm-Crawler crawls programming paradigm from Wikipedia.


##Usage

```bash
$ paradigm-crawler [Language-name] [Root-url(English page)]
```

###Example

```bash
$ paradigm-crawler "C" "https://en.wikipedia.org/wiki/C_(programming_language)"
```
